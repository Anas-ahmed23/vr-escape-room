# VR Escape Room

A virtual reality escape room prototype built using **Unity 6**, **OpenXR**, and the **XR Interaction Toolkit**.

## Features
- XR Origin VR player rig
- Teleportation locomotion
- Snap turning
- Continuous movement
- Meta Quest controller support

## Technologies
- Unity 6
- XR Interaction Toolkit
- OpenXR
- C#
